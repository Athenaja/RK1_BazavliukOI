CosmoStart
Базавлюк Ольга Игоревна, 191-361, варинат 2

Описание игры

CosmoStart - космическая MMORPG. С первых минут игрового процесса предлагается окунуться в мир бесконечного космического пространства. Он затягивает: оригинальное звуковое сопровождение, понятный интерфейс, возможность проводить время с другими игроманами в режиме онлайн.
Большое количество персонажей, заданий, не даст заскучать – здесь всегда есть чем заняться, при желании, можно самостоятельно создавать оружие для своего корабля. Крафт добавляет достоинств Звездным Призракам.
Для большей реалистичности, предусмотрена полноценная рыночная система, в которой разрешается продавать абсолютно все, в том числе оружие – никаких ограничений нет. При достижении пятого уровня, нужно быть готовым встретить космическую вселенную во всей ее жестокости, когда каждый пролетающий мимо корабль может напасть с целью захвата. (Описание для игры взято из интрнета)
